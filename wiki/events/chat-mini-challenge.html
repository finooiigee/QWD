Collect parent/children pairs

Given an array of integers which corresponds to the depth in a traversal path return an array of parent and children node pairs. A node in the return format should be represented by an index into the nodes sorted by order of authority* E.g in the hierarchy: 

     a
   / | \
  b  c  d
 / 
e
The order of authority is a > b > c > d > e

Examples

The hierarchy below has an array of depth: [1,2,3,2,2] by traversing the path: (a) (a -> b) (a -> b -> e) (a -> c) (a -> d) 
     a
   / | \
  b  c  d
 / 
e

The correct result given the argument [1,2,3,2,2] would be: [1,[2,3,4]],[2,[5]]
